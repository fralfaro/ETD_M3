# Actividad: Análisis Exploratorio de Datos (EDA)

## Contexto de la actividad

En esta actividad trabajarás con un notebook en Google Colab que contiene un análisis exploratorio de un conjunto de datos reales.  

El código ya se encuentra completamente implementado. El objetivo de la actividad no es programar, sino analizar, interpretar y comunicar los resultados obtenidos.

El Análisis Exploratorio de Datos (EDA) es una etapa fundamental en proyectos de análisis y ciencia de datos, ya que permite comprender la estructura de los datos, identificar patrones, detectar valores atípicos y apoyar la toma de decisiones.


## Objetivo de Aprendizaje

Al finalizar esta actividad, el estudiante será capaz de:

- Interpretar estadísticas descriptivas básicas.
- Analizar visualizaciones de datos.
- Identificar patrones relevantes y posibles problemas en los datos.
- Comunicar conclusiones basadas en evidencia.


## Material de trabajo

No es necesario modificar el código del notebook.



## Instrucciones

1. Accede al notebook en Google Colab utilizando el enlace proporcionado.

   - Las tablas de estadísticas descriptivas.

## Preguntas de Análisis

1. **Descripción general del conjunto de datos**
   - ¿Cuántas observaciones y variables contiene el conjunto de datos?
   - ¿Qué tipos de variables predominan (numéricas, categóricas)?

2. **Análisis descriptivo**
   - ¿Qué variables presentan mayor dispersión?
   - ¿Se observan valores atípicos? Explica cómo fueron identificados.

3. **Distribuciones**
   - Describe la forma de al menos dos distribuciones.
   - ¿Qué implicancias podrían tener estas distribuciones para el análisis?

4. **Relaciones entre variables**
   - ¿Se identifica alguna relación relevante entre variables?
   - ¿Qué visualizaciones respaldan tu análisis?

5. **Conclusiones**
   - Menciona al menos dos hallazgos relevantes.
   - ¿Qué tipo de decisiones podrían apoyarse a partir de este análisis?

## Criterios de Evaluación

| Criterio | Logrado (100%) | Adecuado (80%) | Básico (60%) | Insuficiente (0%) |

| Comprensión del conjunto de datos | Describe correctamente la estructura, variables y contexto del conjunto de datos | Describe la estructura general con leves imprecisiones | Descripción parcial o poco clara | No describe correctamente el conjunto de datos |
| Interpretación estadística | Interpreta correctamente estadísticas descriptivas y variabilidad | Interpretación mayormente correcta | Interpretación superficial o incompleta | Interpretación incorrecta o ausente |

| Conclusiones y comunicación | Conclusiones claras, coherentes y basadas en datos | Conclusiones adecuadas pero poco profundas | Conclusiones vagas o poco claras | No presenta conclusiones |
